#### Fix: remove unnecessary parens when yielding jsx ([#7367](https://github.com/prettier/prettier/pull/7367) by [@cola119](https://github.com/cola119))

<!-- prettier-ignore -->
```jsx
// Input
function* f() {
  yield <div>generator</div>
}

// Prettier stable
function* f() {
  yield (<div>generator</div>);
}

// Prettier master
function* f() {
  yield <div>generator</div>
}
```
